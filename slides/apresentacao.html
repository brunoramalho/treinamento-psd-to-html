Boa tarde, hoje vamos iniciar o nosso treinamento, o treinamento terá bastante teoria porém muito mais prática.

Como vai funcionar?

Irei apresentar alguns PSDs hoje, vocês escolherão quantos quizerem trabalhar, a idéia é a gente levar os protótipos até o final. O intuito é no final termos um ou mais protótipos implementados para cada um de vocês. Ou seja se algum dia vocês não conseguirem finalizar aqui o que tiver sido proposto é interessante que terminem em casa, pois no outro dia usaremos o mesmo material para dar continuidade ao curso.

Bem, o material encontra-se no meu GitHub [LINK DO PROJETO] façam um fork desse projeto, criem uma branch com o nome de vocês e trabalhem nela, se quiserem ao longo dessas 3 semanas podem ficar mandando pull requests que eu fico analisando o código de vocês a cada etapa do nosso treinamento.
